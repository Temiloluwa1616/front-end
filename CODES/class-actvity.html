DIFFERENCE BETWEEN BLOCK AND INLINE ELEMENT

BLOCK ELEMENT VS INLINE ELEMENT

Block Level Element Inline Level Element
It begins a new line of text Doesn't begin new line of text.Text is placed on the same line
Its width extends beyond the content Its width only extends as far as the inline content
The width and the height values can be set The width and the height can't be set
It can contain text, data,inline or otherblock element It can contain text,data or other inline element
